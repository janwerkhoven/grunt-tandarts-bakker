# tandarts-bakker.nl

### What
The website of a dentist in the Netherlands

### Developed by
Jan Werkhoven, in 2013

### Built with
* Grunt
* Handlebars
* SASS
* jQuery
* Photoshop

### Setup
Requirements:
* Git
* Node
* Grunt

Clone repo, build dist and run server:

1. `cd to/your/folder/`
2. `git clone https://github.com/janwerkhoven/tandarts-bakker.nl.git .`
3. `grunt`
4. Go to http://localhost:9000/
